Hallo Frau Dombach,
ich nutze die IGDB API um Spiele & News auf meiner Seite darzustellen. Dies geschieht per jQuery
& fetch direkt zum Aufruf der Seite.

Die Seite ist manuell als Single Page Application umgesetzt worden, allerdings ohne Routing.
Ich nutze BEM als Naming-Convention und Teile des 7-1 Pattern für die Einteilung der Styles.

Um die Fetches funktional zu bekommen MÜSSEN sie meinen lokalen Cors-Anywhere Proxy Server starten.
Dies machen sie einfach im Terminal mit: node server.js

Das Projekt enthält wie abgesprochen die Meilensteine SASS-Vererbung und SASS-MediaQuerys.
